# sketch

Just some useful commands to start with.

Most commonly used ones:

Sbt:
`$ sbt scalafmtAll`
`$ sbt service/run`
`$ sbt service/assembly`

Docker:
`$ docker stop $(docker ps -a -q)`
`$ docker rm $(docker ps -a -q)`

You might also use:

Docker:
`$ docker rmi $(docker images -a -q)`
`$ docker network prune`
